CS2D 1.0.0.5 @ Steam

15.11.17 08:41:35 pm
CS2D 1.0.0.5 is now live on Steam! Thanks to everyone who made that possible! Special thanks to SQ!
Get CS2D at the CS2D Steam store page!

Dedicated servers or the standalone client (which does not require Steam) can still be retrieved and used the usual way
I recommend using the Steam version as it comes with additional features like over 50 Steam achievements, Steam login etc.

This release is mainly for optimization, bug fixes and Steam specific adjustments. No huge new features.

Changelog
First release with Steam support (Win client only)

Various voice chat issues and stability problems
Various visual lighting engine issues
Various minor visual HUD / tooltip issues
Laser mines rotation
mp_hudscale not behaving correctly in all scenarios
Vertical black boarder (1-4px width) appeared for larger 16:9 resolutions
Graphical glitches when using command line parameters
Large avatars were sometimes not downloaded properly
Client money limit when pickup up gold or coins
Lag animation not looking right under certain network conditions
Committing suicide did not work anymore since 1.0.0.4
Bad projectile timing with increased FPS in 1.0.0.4
Problems with building rotations since 1.0.0.4
Sometimes unable to join servers when they were running Lua tweens
Scoreboard hide HUD option did not hide scripted Lua HUD elements
Possible crash when failing to write advanced radar/minimap cache
Improved stability when server is being DDoSed
Crosshair of spectated player is now semitransparent
When being flashed the radar now also becomes invisible
Max. 2 U.S.G.N./Steam avatars are downloaded & stored concurrently to minimize FPS drops
Lua Command player(id,"mousex") now returns -1 if mouse data is not available
"conkey" (console key) works as intended for more keyboard layouts
Increased fire rate for all weapons (2.5%)
Console now uses scaled text
Scoreboard sorting now takes in account mvp and assists
The scoreboard now shows building count of teammates only
Default windows graphics driver is now OpenGL (for Steam overlay support)
Flash bang effect now hides radar
Fiveseven damage decreased from 19 to 18
Over 50 Steam achievements (Windows only)
Command mp_connectionlimit for max. accepted connections
Command cp_roundend for more additional round information message
Command sb_autosize (automatic scoreboard size)
Command sb_kpd (scoreboard kills / deaths ratio)
Lua Command tween_frame (image frame animation)
"mousemapx" & "mousemapy" parameters for Lua player command (mouse position on map)
Server ticks are displayed in scoreboard and in-game menu (updates per second)
Clients are able to see which operating system server is running
Server lag compensation information in menu
